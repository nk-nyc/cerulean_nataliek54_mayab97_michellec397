## Description

ChoreRulean is an app designed to help users organize their to-do lists. It allows users to make accounts, in which they can create, edit, and delete tasks. These tasks can include multiple users (the owner can invite others), and are sorted by deadline and category (ie. chore, errand, homework, etc.). 

## Install Guide

2. Clone into terminal: in your terminal, write:
```git clone [link] ```
3. Navigate into the repo with ```cd cerulean_nataliek54_mayab97_michellec397```.
4. Install requirements: paste into the terminal:
```pip install requirements.txt```
5. All set! Use the guide below to run it.
